Atrás

Diseño y maquetación web

Los primeros meses de 2023 aproveché mi tiempo aprendiendo los fundamentos del diseño de interfaces y decidí aplicar mis habilidades en un proyecto desde cero. Usando Figma, diseñé un prototipo y luego implementé el diseño utilizando HTML y CSS. Este proyecto me permitió poner en práctica mis habilidades de diseño y programación en un solo proyecto.

Ver la web

Herramientas utilizadas

  • Figma
  • Visual Studio Code
  • HTML
  • CSS (Grid y Flexbox)

Proceso de trabajo

Empecé definiendo cual seria la web y el negocio, y elegí una Inmobiliaria de lujo, en la que en la web se daría información de quienes somos , qué servicios ofrecemos y publicaríamos las viviendas en venta y en alquiler. Una vez definido todo eso pasé a hacer los wireframes en papel de la web empezando por mobile first.

Programación

Una vez terminado el diseño final en alta fidelidad pasé a la parte final de programación en Visual Studio Code, y siendo mi primer proyecto de programación al finalizar el curso de CSS, fueron todo aprendizajes, prueba y error. Puedes ver el código en GitHub pulsando aquí.

Conclusiones del proyecto

Al haber aprendido HTML y CSS recientemente, diría que han mejorado muchisimo mis habilidades de diseño UI, ya que he tenido siempre en cuenta el cómo programaría cada elemento, y también tenia en cuenta de que esos elementos tenía que diseñarlos para luego poder hacerlos responsive de manera eficiente.

Gracias por tomarte tu tiempo para ver este proyecto!
